# Vision 기반 델타 로봇 자동 분류 시스템

## 프로젝트 개요
본 프로젝트는 컨베이어 벨트를 통해 이송되는 나사 및 볼트와 같은
소형 부품을 카메라 기반 머신 비전으로 인식하고,
불량품과 정상품을 판별한 뒤 델타 로봇을 이용하여
규격에 따라 자동으로 분류하는 시스템을 구현하는 것을 목표로 합니다.

산업 현장에서 사용되는 자동화 라인 구조를 모델로 하여,
비전 인식, 품질 검사, 로봇 제어가 연동되는
통합 자동 분류 시스템을 설계·구현하는 데 중점을 둡니다.

## 시스템 구성
- 컨베이어 벨트를 통한 부품 이송
- 카메라를 이용한 부품 촬영
- 이미지 처리 기반 불량 판별 및 규격 분류
- 델타 로봇을 이용한 픽 앤 플레이스(Pick & Place)
- 정상 제품과 불량 제품의 분리 처리

## 대상 하드웨어

### 1. 제어 및 연산부 (핵심 두뇌)
- **BeagleBone Black Rev. C** : 영상 처리 및 전체 시스템 상위 제어용 (비전 파이프라인, UI/로직, 로깅 등)
- **STM32F407 Discovery Board** : 실시간 모터 제어 및 하드웨어 가속 담당(모터 드라이버 인터페이스, 엔코더/리미트 처리)
- **CNC Shield V3 + A4988 드라이버 (×4)** : Nema17 스테퍼 모터 구동용

### 2. 구동 모터부
- **NEMA 17 스테퍼 모터 (×4)** : 델타 로봇(3축)용 3개 + 컨베이어용 1개
- **MG995 서보 모터 (×1)** : 흡착용 주사기 구동 그리퍼(서보 기반 리프팅/각도 제어)

### 3. 델타 로봇 프레임 (팔 및 지지대)
- **15mm 목재 봉(15T)** : 기둥 및 지지대용 (절단: 220 mm ×3개, 330 mm ×3개)
- **M4 × 250 mm 나사산 봉 (×6)** : 로봇 팔(Link) 구성
- **M4 로드 엔드 (직선형, ×12)** : 팔의 관절 부품(각도형 아님 — 직선형 사용)
- **M4 너트 (12개)** : 로드 엔드 고정 및 길이 미세 조절용
- **625ZZ 볼베어링 (내경 5 mm, ×3)** : 모터 축 양단 지지용

### 4. 컨베이어 및 벨트 구동 시스템
- **GT2 폐쇄형(Closed-loop) 벨트 (1220 mm, ×1)** : 컨베이어 본체용
- **GT2 개방형(Open-end) 벨트 (3–5 m)** : 로봇 각 축 구동용 예비 길이
- **GT2 30T 풀리 (보어 5 mm, ×1)** : 컨베이어 모터 연결용
- **케이블 타이(타이랩)** : 벨트 고정 및 전선 정리

### 5. 흡착 시스템 (그리퍼)
- **ZPT06BSJ6-B5-A8 흡착 패드** : 물체 접촉부(재질/크기 확인 필요)
- **PC4-M5 공압 피팅** : 패드와 호스 연결
- **4 mm PU 에어 호스** : 진공 통로
- **10 ml 주사기** : 단순 진공(흡착) 발생원(테스트용; 소형 진공펌프 대체)

### 6. 센서 및 기타 소모품
- **마이크로 리미트 스위치 (×3)** : 델타 로봇의 원점(Homing) 확인용
- **USB 웹캠** : BeagleBone Black 연결용 비전 센서(카메라 사양은 해상도/프레임 고려)
- **전원 어댑터 12V 5A** : 시스템 전체 전원(모터 및 보드 전력 요구량 확인 권장)
- **MDF / 합판 베이스** : 전체 장치 고정용 베이스 플레이트
